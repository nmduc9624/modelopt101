# TinyML Chat *(planned)*

A from-scratch LLM chat application that runs on a personal **laptop** (no Colab),
demonstrating the optimization ideas from the exercises applied to a real model.

Follows:
- [TinyChatEngine](https://github.com/mit-han-lab/TinyChatEngine) - the full application.
- [tinychat-tutorial](https://github.com/mit-han-lab/tinychat-tutorial) - the exercise version.

**Requirements:**
- No need to run on Colab.
- Self-contained; can be installed and run on a personal laptop.

> Not built yet - to be implemented after the core exercises.
